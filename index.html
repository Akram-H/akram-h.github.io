Mina Game Engine

The Python based game engine from the stars.

About

Mina is based on Python and the first game engine that was created with Python and has an editor. The editor makes it easy to manage objects and create games in general.

Mina is completely free and open source. The code is published under the MIT license. The games you will make are fully yours.

Download

Mina is currently only available for windows, but we are working hard to release on other platforms
